# Wilderkin League — Marketing Automation

Marketing automation for [Wilderkin League](https://wilderkinleague.com), a free pixel art auto-battler.

## About the Game

- 10 animal species, 95+ abilities, deep synergy system
- Free, browser-based, no download or signup needed
- Pixel art aesthetic

**Play at: https://wilderkinleague.com**

## Scripts

- `reddit_poster.py` — Posts to r/autobattler, r/indiegaming, r/WebGames etc.
- `mastodon_poster.py` — Posts to Fediverse instances
- `hn_poster.py` — Show HN / Ask HN submissions
- `bluesky_poster.py` — Bluesky gaming content
- `devto_poster.py` — Dev.to articles
- `seo_content_generator.py` — Generates SEO landing pages
- `run.sh` — Runs all channels, designed for 30-min cron

## Usage

```bash
bash run.sh
```
